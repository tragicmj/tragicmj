### 🔭 What I do

- Own the **React Native mobile app** for Book An Artist end-to-end — architecture, releases, native tooling (Xcode/CocoaPods, Android build config), and performance
- Lead **creator-facing features** across the Wescover web platform (`wescover-edit`) — product editor, pricing, and analytics
- Work across the **full stack**: React/Redux, GraphQL/Apollo on the frontend; Node/Express APIs; MongoDB on the backend
- Drive platform modernization — recent work includes a **Craco → Vite migration**, **React Native 0.74 → 0.85 upgrades**, and Core Web Vitals improvements

### 🌱 Currently exploring

- Apple Silicon dev environment tooling and Android 16KB page-size alignment for React Native
- Multi-region pricing & currency architecture for e-commerce platforms
- Payout infrastructure (Stripe Connect, international payout options)

### 💬 Ask me about

React Native, React/Redux (including Redux Form & recompose-era codebases), GraphQL/Apollo, Node.js, MongoDB, e-commerce architecture, CI/CD pipelines, and shipping features on legacy codebases without breaking them.
